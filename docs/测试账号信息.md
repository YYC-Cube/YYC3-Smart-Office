## 环境变量配置（开发/测试）

```env
# 必须配置
JWT_SECRET=yyc3-dev-jwt-secret-key-for-testing-only-min-32-chars
DATABASE_URL=postgresql://postgres:password@localhost:5432/yyc_smart_office

# 可选配置
PORT=3029
NODE_ENV=development
CORS_ORIGIN=http://localhost:3029
LOG_LEVEL=debug
```

## 快速启动

```bash
# 安装依赖
pnpm install

# 启动开发服务器（端口 3029）
pnpm dev -p 3029

# 运行测试
pnpm test

# 构建验证
pnpm build
```

## API 测试端点

| 端点 | 方法 | 说明 |
|------|------|------|
| `/api/health` | GET | 健康检查 |
| `/api/auth/csrf-token` | GET | 获取 CSRF 令牌 |
| `/api/auth/login` | POST | 登录认证 |
| `/api/auth/status` | GET | 认证状态检查 |
| `/api/auth/logout` | POST | 退出登录 |
| `/api/auth/refresh` | POST | 刷新令牌 |
| `/api/auth/register` | POST | 用户注册 |
| `/api/auth/verify-env` | GET | 环境变量验证 |
| `/api/db/status` | GET | 数据库状态检查 |
| `/api/permissions/check` | GET | 权限校验 |
